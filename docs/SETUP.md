# Development Environment Setup Instructions

## Frontend Setup
1. **Clone the repository:**  
   ```bash
   git clone https://github.com/taharat/taharat-todoapp.git
   cd taharat-todoapp/frontend
   ```  
2. **Install dependencies:**  
   ```bash
   npm install
   ```  
3. **Start the frontend application:**  
   ```bash
   npm start
   ```

## Backend Setup
1. **Navigate to the backend directory:**  
   ```bash
   cd taharat-todoapp/backend
   ```  
2. **Install dependencies:**  
   ```bash
   npm install
   ```  
3. **Start the backend server:**  
   ```bash
   npm run dev
   ```

## AI Module Setup
1. **Navigate to the AI modules directory:**  
   ```bash
   cd taharat-todoapp/ai-modules
   ```  
2. **Install dependencies:**  
   ```bash
   pip install -r requirements.txt
   ```  
3. **Run the AI models:**  
   ```bash
   python main.py
   ```

## Conclusion
After completing these steps, you should have the development environment for the **taharat-todoapp** set up for frontend, backend, and AI modules.
